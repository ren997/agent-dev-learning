# Agent 开发术语速查

这份速查卡用于复习 Agent 开发中的核心词汇。后续课程会持续补充。

## Agent

由大模型驱动的程序，能根据目标选择下一步行动，并通过工具和反馈推进任务。

## LLM

Large Language Model，大语言模型。它负责理解自然语言、生成文本、判断是否需要调用工具。

## LLM 应用形态

基于大语言模型做出来的应用方式，例如 Chatbot、Workflow 和 Agent。LLM 是底层能力，应用形态是开发者把 LLM 接入产品后的组织方式。

## Chatbot

聊天问答型应用。它通常接收用户输入，然后直接生成回答。普通 Chatbot 不一定是 Agent，除非它能规划、调用工具并根据反馈继续执行。

## Workflow

固定流程型应用。开发者提前定义好步骤，模型只在某些步骤中完成生成、分类、总结等任务。

## Prompt

给模型的指令，包括角色、任务、规则、输出格式和限制。

## Tool / Function Calling

模型请求调用外部函数或 API 的机制。模型通常只生成工具名和参数，真正执行工具的是你的应用代码。

## Agent Loop

Agent 的执行循环：观察状态，决定行动，调用工具或回答，接收反馈，然后决定是否继续。

## State

Agent 当前任务的状态，例如历史消息、工具结果、用户目标、学习进度和中间结论。

## RAG

Retrieval-Augmented Generation，检索增强生成。先从资料库检索相关内容，再让模型基于资料回答。
