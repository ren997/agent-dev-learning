# Teaching Notes

- 用户想学习 Agent 开发，目标包括做自己的 Agent 应用，以及提升工作、转岗或面试中的 AI 应用开发能力。
- 用户已有后端开发基础。
- 优先使用 Python。
- 学习节奏偏每天 30 到 60 分钟，小步推进。
- 后续每次学习应尽量留下可复习文件：课程、练习代码、学习记录或参考资料。
- 课程和参考资料默认使用 Markdown，方便在 Cursor 中阅读、编辑和长期维护。只有用户明确需要交互页面、打印版或可视化展示时，才生成 HTML。
- 用户在第 003 课反馈“内容空洞，不能深刻理解”。后续讲解工具调用、记忆、RAG 时，要优先使用具体执行过程、角色分工、状态变化和最小例子，少用只停留在概念层的表述。
