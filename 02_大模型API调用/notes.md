# 大模型 API 调用笔记

## 我学到的概念

### API Key

API Key 是调用模型服务的凭证。不能写死到代码里，应该通过环境变量读取。

### client

SDK 客户端，负责向模型服务发送请求。

### model

指定要调用的模型。

### messages

对话上下文。常见角色有：

- `system`：定义模型身份、规则和边界
- `user`：用户输入
- `assistant`：模型历史回答

### response

模型返回结果。程序需要从 response 中取出文本内容。
