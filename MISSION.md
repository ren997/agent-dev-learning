# Mission: Agent 开发

## Why

我想系统学习 Agent 开发，目标是能用 Python 做出自己的 Agent 应用，例如学习助手、代码助手或工作自动化工具。这个能力也要能帮助我在工作、转岗或面试中展示 AI 应用开发能力。

## Success looks like

- 能清楚解释 Agent、普通聊天机器人和固定工作流的区别
- 能用 Python 调用大模型 API，完成连续对话
- 能手写一个最小 Agent，让模型根据任务决定是否调用工具
- 能加入简单记忆，让 Agent 保存和读取学习进度
- 能做一个小型 RAG 问答功能
- 能使用一个主流 Agent 框架重构项目
- 能完成一个可演示的个人学习助手 Agent

## Constraints

- 学习节奏：每天 30 到 60 分钟，小步学习和练习
- 编程基础：已有后端开发经验，可以直接使用项目结构、API、数据库等概念
- 技术路线：优先使用 Python
- 教学方式：每次学习要留下文件记录，方便下次继续

## Out of scope

- 暂时不深入训练大模型或微调模型
- 暂时不追求复杂多 Agent 系统
- 暂时不做大型前端界面，先以命令行和简单 Web 页面为主
